Definitie

Random Access Memory, sau Ram, este o componenta a calculatorului care stocheaza date temporar, servind ca memorie "de lucru" si care se fixeaza in sloturile de memorie de pe placa de baza. RAM-ul este de obicei asociat cu DRAM, care este un tip de memorie RAM. Deoarece informatiile sunt accesate aleatoriu in loc de secvential, cum ar fi pe un CD sau pe un hard disk, calculatorul poate accesa datele mult mai rapid. Cu toate acestea, spre deosebire de ROM sau hard disk, memoria RAM este o memorie volatila si necestia putere pentru a pastra datele accesibile. Daca calculatorul este orpit, toate datele stocate in memoria RAM sunt pierdute.

Tipuri de RAM

Pe parcursul evolutiei calculatorului s-au inventat diferite variatii ale memoriei RAM. Unele din cele mai comune exemple sunt DRAM (Dynamic RAM), SRAM (Static RAM), Flash Memory, SDRAM (Synchronous dynamic RAM), DDR SDRAM (Double data rate SDRAM), RDRAM (Rambus dynamic RAM).

Imaginea din dreapta este un exemplu de DRAM.

Informatii aditionale

După încărcarea sistemului de operare, fiecare program pe care îl deschideți, cum ar fi browserul pe care îl utilizați pentru a vedea această pagină, este încărcat în memorie în timp ce rulează. Dacă sunt deschise prea multe programe, calculatorul va schimba datele din memorie între memoria RAM și HDD.

Viteza si performanta calculatorului sunt in mare parte atribuite cantitatii de memorie din calculator. Daca calculatorul nu are suficienta memorie pentru a rula sistemul de operare sau programele software utilizate, performanta acestuia va scadea. Mai multa memorie intr-un calculator permite acestuia sa ruleze mai eficient sistemul de operare si orice program software, reducand sau eleminand necesitatea ca fisierele si datele sa fie introduse sau scoase din memorie in fiecare secunda.

Istorie

Primul RAM a aparut in 1947 si utilizeaza tubul William. Acesta utiliza un tub cu raze catodice (CRT) si datele erau stocate pe fata CRT-ului ca spoturi incarcate electric.

Cea de-a doua forma de RAM utilizata pe scara larga a fost memoria miez-magnetica, inventata in 1974. Aceasta memorie functioneaza prin utilizarea unor inele metalice mici si fire care se conecteaza la fiecare inel. Un singur bit de data putea fi stocat pe fiecare inel si accesat in orice moment.

Cu toate acestea, memoria RAM pe care o stim astazi, a fost inventata in 1968 de catre Robert Dennard si este cunoscuta in mod specific ca dynamic random access memori, sau DRAM. Pentru stocarea bitilor se foloseau tranzistori.
